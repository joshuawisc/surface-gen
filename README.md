# Manifold Visualizer

## Setup and run
1. Install flask - `pip install flask`
2. Install python packages - `pip install seaborn tqdm numpy matplotlib gunicorn networkx cvxpy`  
  a. cvxpy  
  b. networkx  
  c. gunicorn  
  d. matplotlib  
  e. numpy  
  f. tqdm  
  g. seaborn   
3. Run `flask run` in the project source folder
4. Go to http://localhost:5000/
